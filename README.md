# Анализ качества вин - Отчет по исследованию

## Описание проекта
Разработка модели машинного обучения для классификации качества вин на основе физико-химических показателей.

## Результаты
- Точность модели: 97.2%
- Улучшение: +17.3% относительно базовой модели
- Методология: CRISP-DM

## Структура отчета
1. Постановка задачи
2. Бизнес-ценность
3. Метрики качества
4. EDA-анализ
5. Методика решения
6. Результаты и рекомендации

## Технологии
- Python 3.12.7
- Scikit-learn
- Pandas, NumPy
- Matplotlib, Seaborn
